## Run on local machine

Firstly install the dependencies: 

```bash
npm install
# or
yarn install
```

Then run the following command to start development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
